# Shadow Compile AI

Shadow Compile AI is a lightweight intelligent debugging and performance analysis engine.

## Features
- Executes user Python code
- Captures runtime errors
- Measures execution time
- Profiles peak memory usage
- Generates structured analysis report

## How It Works
1. User pastes Python code.
2. System executes in controlled environment.
3. Runtime metrics are collected.
4. Errors and performance stats are displayed.

## Future Scope
- Intelligent error classification
- Optimization suggestions
- Hardware-aware code recommendations
- Web-based interactive dashboard

## Run Instructions
Install dependencies:
pip install memory-profiler

Run:
python app.py
